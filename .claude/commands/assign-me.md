# Assign Me Command

Jira 이슈에 자신을 담당자로 할당합니다.

## 사용법
```
/assign-me <issue-key>
```

예시:
```
/assign-me PROJ-123
```

## 수행 작업

1. 인자로 받은 이슈 키가 유효한지 확인

2. mcp-atlassian을 사용하여:
   - 해당 이슈의 현재 상태 조회
   - 이슈의 담당자를 현재 사용자로 할당
   - 이슈 상태가 "To Do"인 경우 "In Progress"로 변경

3. 결과를 다음 형식으로 출력:

```
✅ 이슈 할당 완료

**이슈**: [PROJ-123] 이슈 제목
**담당자**: [현재 사용자] (할당됨)
**상태**: To Do → In Progress

🔗 [Jira에서 보기](이슈 URL)

💡 이제 이 이슈를 작업할 수 있습니다!
```

4. 오류 처리:
   - 이슈 키가 존재하지 않는 경우: "이슈를 찾을 수 없습니다"
   - 이미 다른 사람에게 할당된 경우: 경고 메시지와 함께 진행 여부 확인
   - 권한이 없는 경우: "이슈를 할당할 권한이 없습니다"
