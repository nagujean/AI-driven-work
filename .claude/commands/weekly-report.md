# Weekly Report Command

주간 보고서를 생성하여 팀 미팅에서 공유합니다.

## 수행 작업

1. mcp-atlassian을 사용하여 이번 주 데이터 조회:
   - 이번 주에 내가 완료한 이슈 (Done 상태로 변경된 이슈)
   - 현재 진행 중인 이슈 (In Progress)
   - 다음 주에 시작할 예정인 이슈 (To Do 또는 미할당)

2. 주간 보고서를 다음 형식으로 생성:

```
# 📊 주간 업무 보고서
**기간**: [시작일] - [종료일]
**작성자**: [사용자 이름]

## ✅ 이번 주 완료 사항
- [PROJ-100] 이슈 제목
  - 주요 작업 내용 요약
- [PROJ-101] 이슈 제목
  - 주요 작업 내용 요약

## 🚧 진행 중인 작업
- [PROJ-200] 이슈 제목 (진행률: 60%)
  - 현재 상황
  - 예상 완료일

## 📅 다음 주 계획
- [PROJ-300] 이슈 제목
  - 예상 작업 내용

## 🎯 주요 성과
- [중요한 성과나 학습 내용]

## 🚨 이슈 및 블로커
- [문제가 있다면 기술]
- [도움이 필요한 부분]

## 💭 소감 및 개선 사항
- [이번 주 업무에 대한 소감]
- [다음 주 개선하고 싶은 점]
```

3. 생성된 보고서를 Confluence에 저장할지 물어보기:
   - 저장 위치: [프로젝트 Space]/주간 보고서/[날짜]
   - 사용자가 원하면 Confluence 페이지 생성
